# Country Caravan Care

Country Caravan Care site

## Structure

- `index.html` - production page served by GitHub Pages.
- `assets/css/` - site stylesheets.
- `assets/js/` - JavaScript files.
- `assets/images/` - image and favicon assets.
- `assets/data/` - structured site data, such as reviews.
- `dev/` - ignored local experiments and test pages.
